## お題

```
入力として1から100までの整数を受け取れること。それ以外は、例外を返すこと。
受け取った数値をそのまま返すこと。
ただし3の倍数のときは数の代わりに「Fizz」と、5の倍数のときは「Buzz」と返し、3と5両方の倍数の場合には「FizzBuzz」を返すこと。
```
